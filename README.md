# H2O Gyorsszervíz – Vízszerelő gyorsszolgálat weboldala

Ez a projekt a H2O Gyorsszervíz hivatalos weboldala, amely Budapesten és Pest megyében nyújt professzionális vízszerelési szolgáltatásokat.

## Főbb jellemzők
- Gyors kiszállás Budapesten és Pest megyében
- 0–24 órás elérhetőség, hétvégén és ünnepnapokon is
- Csaptelep csere és javítás
- Vízvezeték javítás, csere, szerelés
- Szifoncsere, lefolyócsövek javítása
- Csőtörés bemérése és javítása modern technológiával
- Vízszivárgás felderítése kamerás vizsgálattal
- Szelep, flexibilis cső, sarokszelep csere
- WC tartály javítás, csere
- Fürdőszobai és konyhai vízszerelés
- Mosógép, mosogatógép bekötés
- Teljes vízvezeték rendszer felújítása
- Fagyás miatti csőrepedés javítása
- Tiszta, precíz munkavégzés, garanciával

## Technológia
- Next.js
- React
- Tailwind CSS

## Telepítés
1. Klónozd a repót:
   ```bash
   git clone <repo-url>
   ```
2. Telepítsd a függőségeket:
   ```bash
   npm install
   ```
3. Indítsd el a fejlesztői szervert:
   ```bash
   npm run dev
   ```
